# Unityによる3DCGプログラミング - 基本原理と実践 -
本リポジトリは，コロナ社『[Unityによる3DCGプログラミング - 基本原理と実践 -](https://www.coronasha.co.jp/np/isbn/9784339013825/)』の書籍に対応したUnityプロジェクトです。
書籍の各章で紹介しているサンプルシーンと，演習問題の解答例コードを収録しています。


## 動作環境
本プロジェクトは以下の環境での動作を想定しています。
- Unity 6 (`6000.3.11f1`)
- Windows 11

他の環境やPCのスペックによっては正常に動作しない場合があります。


## 環境構築の手順
### 1. Unityのインストール
Unity Hub を起動し，「インストール」タブからUnity 6000.3.11f1をインストールしてください。
バージョンが一覧に表示されない場合は，[Unity ダウンロードアーカイブ](https://unity.com/releases/editor/archive) から該当バージョンをインストールできます。

### 2. プロジェクトを開く
本リポジトリをクローンまたはダウンロードした後，Unity Hub の「プロジェクト」タブから「開く」を選択し，リポジトリのルートフォルダを指定してください。
初回起動時にはアセットのインポート処理が行われるため，開くまでに時間がかかることがあります。


## プロジェクト構成
```
Assets/
├── Chapters/          # 書籍の各章に対応するサンプルシーン
│   ├── Chapter1/
│   ├── Chapter2/
│   ├── Chapter3/
│   ├── Chapter4/
│   ├── Chapter5/
│   └── Chapter6/
└── Exercises/         # 演習問題の解答例コード
    ├── Exercise1/
    ├── Exercise2/
    ├── Exercise3/
    ├── Exercise4/
    ├── Exercise5/
    └── Exercise6/

Exercises.pdf           # 演習問題（PDF）
```

### [Exercises.pdf](Exercises.pdf)
演習問題の問題文をまとめたPDFファイルです。
リポジトリのルートディレクトリに配置されています。

### Chapters
書籍の各章に対応するサンプルシーンが含まれています。
各シーンを開くと，書籍内で提示している画像と同じ結果をUnityエディタ上で確認できます。
書籍を読みながらシーンの構成やスクリプトなどを適宜参照してください。

### Exercises
各章の演習問題に対する解答例のコードが含まれています。
演習問題に取り組んだ後，自分の解答と比較する際に参考にしてください。
